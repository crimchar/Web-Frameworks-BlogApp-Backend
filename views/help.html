Routes
GET '/' Displays this help page with all of the different routes
GET '/Users' Displays a list of all of the users
POST '/User' Creates a new user with the following properties: UserID, First Name, Last Name, Email Address, and Password, all fields are required
GET '/User/{ID}' Returns the user with the specified ID
PATCH '/User/{ID}' Updates the user with the specified ID
DELETE '/User/{ID}' Deletes the user with the specified ID
